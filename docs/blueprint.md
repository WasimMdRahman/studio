# **App Name**: ParkWise

## Core Features:

- Slot Status Dashboard: Real-time display of parking slot availability (Green, Red, Gray).
- Slot Booking: Booking functionality for available slots with user ID, timestamp stored in Firestore.
- Booking Cancellation: Cancellation functionality to release booked slots.
- Slot Auto-Release: Automated expiry and release of unused slots.
- User Authentication: Firebase Authentication for user login/signup.
- AI Parking Tips: Use a generative AI tool to generate parking tips based on current user behavior. This is to give the users guidance around finding available parking
- Slot Status Details: Display user booked and available slots

## Style Guidelines:

- Primary color: Soft blue (#7EC8E3) to represent calmness and reliability.
- Background color: Light gray (#F0F4F7) to provide a clean, neutral base.
- Accent color: Subtle green (#A8D8B9) to highlight available slots.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable interface.
- Use clear icons for slot status: car icon for booked, checkmark for available, clock icon for expired.
- Card-based grid layout for slots, responsive for mobile and desktop.
- Subtle animations for booking confirmations and status updates, including eye-catching car parking animations.
